# 리눅스 기본 명령
리눅스를 사용하기 위한 기본 명령어들을 학습합니다.

## 도움말
`--help`옵션 알아보기

## 터미널 및 명령 관리
* clear : 화면정리 지우기, 단축키 `ctrl`+`l`
* history : 명령어 기록

## 시스템 상태 확인하기
* who : 누가 접속했는지 확인
* whoami : 
* w : 사용자의 작업 확인
* users : 접속한 사용자
* arch : 아키텍쳐 확인
* logname :
* uname :
* hostname :
* id

## 달력 및 시간
* cal
* date


## 파일 및 디렉터리
* ls : 디렉터리 및 파일의 목록을 확인합니다.
* pwd : 현재 디랙터리의 경로를 출력합니다.



* [wc](/command/wc) : 파일의 문자수 세기
