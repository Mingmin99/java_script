# grep
Globally find Regular-Expression and Print로 지정된 표현형식이 전체에 있는지 찾아서 프린트 합니다.

## 패턴매팅

```
grep pattern 파일명
```
주어진 패턴이 있는지 해당 지정된 파일에서 검색합니다.

## ~ | grep 패턴
파이프를 통하여 결과를 이용하여 지정된 패턴의 검색 결과를 찾는 방법입니다.

## 주요옵션

* w : 전체 단어가 일치되는지 
* n : 라인 넘버 출력
* v : 단어가 일치하지 않는 경우 출력
* l : 해당 파일명을 출력
