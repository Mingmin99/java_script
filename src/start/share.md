# 다중 사용자의 자원 공유
리눅스에서 여러 사용자들이 자원을 공유하여 사용하는 이유는 자원의 효율적인 활용과 비용 절감을 위해서입니다. 여러 사용자들이 동시에 사용할 수 있는 자원은 불필요한 중복 구매나 낭비를 줄일 수 있고, 자원을 더욱 효율적으로 활용하여 생산성을 높일 수 있습니다. 또한, 사용자들 간에 협업을 통해 작업을 더욱 효율적으로 수행할 수 있습니다.  



## 비용 절감
여러 사용자들이 동시에 사용 가능한 자원은 `비용`을 절감할 수 있습니다. 예를 들어, 여러 사용자가 사용하는 프린터는 중복 구매를 막고, 하나의 자원을 `공유`함으로써 비용을 절감할 수 있습니다.  



![image-20230416152937776](./img/image-20230416152937776.png)



## 효율적인 활용
자원의 효율적인 활용을 위해 여러 사용자가 공유할 수 있는 자원은 더욱 효율적으로 사용될 수 있습니다. 예를 들어, 여러 사용자가 공유하는 서버의 `CPU`나 `메모리` 등의 자원은 각각의 사용자가 개별적으로 사용하는 것보다 더욱 효율적으로 사용될 수 있습니다.

## 협업을 위한 자원 공유
사용자들 간의 협업을 위해 자원을 공유합니다. 예를 들어, 여러 사용자들이 공유하는 `파일 서버`는 여러 사용자들이 문서를 공유하고, 동시에 편집할 수 있도록 합니다. 이러한 자원 공유를 통해 사용자들은 서로 협업하여 작업을 더욱 효율적으로 수행할 수 있습니다.  



## 접근제어
자원 공유는 사용자들에 대한 접근 제어를 더욱 효율적으로 관리할 수 있도록 합니다. 사용자 그룹을 구성하여 각 그룹별로 자원에 대한 권한을 설정함으로써, 보안을 강화하고, 자원 관리를 더욱 효율적으로 수행할 수 있습니다.
